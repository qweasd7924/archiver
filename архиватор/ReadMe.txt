Архиватор работает только в системе Windows, не поддерживает Unix из - за обработки строк, не поддерживает коментарии и 
папки.

При вводе в консоль всегда сначала необходимо ввести ссылку на новый или сущестующий архив с учетом 
расширения ("D:\\newFolder\\newZipArchiv.zip").

Для создания архива после нужно ввести через пробел ссылки на входящие файлы с учетом расширения
файла и без пробелов в ссылках(D:\\newFolder\\newZipArchiv.zip   D:\\aaa.txt   C:\\pict\\pict1.jpg). 

Для распаковки архива необходимо ввести команду unpack_to , а затем ссылку на распаковку.
(D:\\ZipFileInput.zip unpack_to E:\\aaa\\bbb\\);

Для добавления в архив одного файла ввести после ссылки на архив команду "add",а затем ,после пробела, ссылку на добавляемый объект с учетом расширения
(D:\\ZipFileWithAddingElement.zip  add  D:\\ccc.txt);

Все входные файлы не должны содержать в имени "unpack_to" и пробелы.
